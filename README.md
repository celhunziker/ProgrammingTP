## 2. Project Description & Modules
This project consists of two main parts: **Data Collection** (Web Scraping) and **Data Manipulation & Visualization**. Our objective is to extract the top-rated movies across multiple genres from Filmaffinity and analyze their underlying characteristics to answer specific analytical questions.

The codebase is split into the following main modules/files:

### `scraper.py`
Module corresponding to Part 1 of the assignment. It uses `requests` and `BeautifulSoup` to statically scrape the top movies of 5 different genres (Drama, Comedy, Thriller, Sci-Fi, Horror) from Filmaffinity ranking pages. It also navigates internally into actor profile pages to scrape biographical information (such as their country of birth).

### `analysis-v2.py`
Module corresponding to Part 2 of the assignment. It consumes the data retrieved by the scraper and performs heavy data wrangling operations using `pandas` (handling missing values, data type conversion, concatenation, discretization, and aggregating via groupby). Finally, it answers the proposed analysis questions by producing an interactive HTML dashboard using `plotly`.

### `data/` *(Folder)*
Directory that is automatically generated by the scraper. It contains the resulting tabular `.csv` files (one resource per genre: `filmaffinity_drama.csv`, etc.) and an additional `actores.csv` mapping the actors' origins.

### `plots/` *(Folder)*
Directory generated by the analysis script where all the static PNG exports of our visualizations and the interactive HTML Dashboard are stored.

## 3. Package Dependencies
The project requires **Python 3** and the following external packages:
- `requests`
- `beautifulsoup4`
- `pandas`
- `numpy`
- `plotly`
- `cloudscraper`
- `kaleido` *(Required by Plotly to export static PNG images)*

*(You can install them directly using your environment via: `pip install -r requirements.txt`)*

## 4. How to Execute the Code
To run the full pipeline, please execute the scripts in the following order from the root directory of the project using your terminal:

**Step 1: Execute Data Collection**
```bash
python scraper.py
```
> **Note:** It lasts a couple of minutes while it crawls through the genres and parses the biographical actor pages located in the Filmaffinity server.

**Step 2: Execute Data Analysis and Dashboard Generation**
```bash
python analysis-v2.py
```
After Step 2 finishes, the code will output an `interactive_dashboard.html` inside the `plots/` folder. You can simply double-click it to open it in any modern web browser and explore the results interactively.
